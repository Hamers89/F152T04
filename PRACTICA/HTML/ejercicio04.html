Posiciones en CSS

Esta caja tiene posición static. Es el valor por defecto y no se puede mover con top, left, right o bottom.
Esta caja tiene posición relative. Puedes moverla usando top, left, right o bottom, pero mantiene su espacio original.
Esta caja tiene posición absolute. Se posiciona respecto al ancestro posicionado más cercano o al body si no hay ninguno.
Esta caja tiene posición fixed. Permanece fija en la ventana del navegador aunque hagas scroll en la página.
Esta caja tiene posición sticky. Se comporta como relative hasta que se alcanza un punto de scroll y luego se fija.
